## 현재 브랜치를 찾는 방법
- git branch --show-current 명령의 결과를 현재 브랜치로 취급한다.

## 분기점을 찾는 방법
- git merge-base 명령을 사용해서 현재 브랜치와 모든 브랜치 간의 공통 조상 커밋 ID를 찾는다.
- 결과로 받은 커밋들과 현재 브랜치의 HEAD 사이의 커밋 개수가 0이 아닌 것 중 가장 개수가 적은 커밋을 분기점으로 취급한다.

## 브랜치 범위 정의
- main 브랜치에서 HEAD까지가 아닌 위 규칙대로 찾은 현재 브랜치의 분기점에서 HEAD까지의 커밋이다.
- 분기점은 제외하고 HEAD는 포함한다.
