# CSharp-Dev
C# 개발에 유용한 환경을 구성합니다. 추가 되는 항목은 다음과 같습니다.

## MCP Servers
### microsoft-learn
Micorosoft Learn에 직접 접근하여 공식 제공 되는 예제를 확인할 수 있게 합니다.

## Additional Contexts
### [Project Exploration](hooks/contexts/project-exploration.md)
Project 경로를 찾을 때 기본적으로 단순 검색을 사용하기 때문에 규모가 큰 저장소에서는 시간이 오래 걸립니다. 이를 방지하기 위해 Solution 파일에서 경로를 찾도록 지침을 추가합니다.

### Requirements
`bash` 명령을 사용할 수 있어야 합니다. Windows 환경이라면 환경 변수에 `%Git%/bin` 경로를 추가하세요.  
일반적으로 `C:\Programfiles\Git\bin` 경로를 추가하면 됩니다.
